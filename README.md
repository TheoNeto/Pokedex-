# Pokedex

Personal project of a **web-based Pokédex** developed with **HTML, CSS, and JavaScript**, consuming data from the **PokéAPI** to display Pokémon information dynamically and responsively.

## Description

This application allows users to view Pokémon and their main data, demonstrating the use of **REST API consumption**, **DOM manipulation**, and **modern JavaScript** in a simple and functional frontend project.

The project was built with a focus on learning, code organization, and frontend best practices.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Fetch API  
- PokéAPI  

## Features

- Pokémon listing  
- Display of basic information (name, image, type, etc.)  
- External API consumption  
- Responsive interface  
- Dynamic content updates
